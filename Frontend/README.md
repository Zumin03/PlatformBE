# Measurement Platform UI

Frontend application built with React, TypeScript, and Vite.

## Installation

1. Clone the repository.
2. Install dependencies:

```bash
npm ci
```

3. Start the development server:

```bash
npm run dev
```

4. Create a production build:

```bash
npm run build
```

5. Run lint:

```bash
npm run lint
```

## Libraries Used

- [React Router](https://reactrouter.com/) - Handles client-side routing, allowing navigation between pages without full page reloads.
- [TanStack Query](https://tanstack.com/query/latest) - Manages server state, data fetching, caching, and background updates.
- [MUI](https://mui.com/) - A React UI component library that provides ready-to-use, customizable interface components.

## Scripts

- `npm run dev` - starts the development server
- `npm run build` - creates the production build
- `npm run lint` - runs lint checks
- `npm run preview` - previews the built application locally
